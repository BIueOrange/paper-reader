# Source: DOI / arXiv identifier

The user provides a bare DOI or arXiv ID.

1. Resolve the identifier with an available browser or HTTP fetch tool:
   - DOI: `https://doi.org/{DOI}` to locate the paper page.
   - arXiv: `https://arxiv.org/abs/{ID}` to locate available formats.
2. Capture bibliographic metadata and determine the available PDF, HTML, or source version.
3. Prefer a lawful open-access version when the version of record is paywalled; record exactly which version was read.
4. Re-route to the appropriate source fragment (pdf-text, scanned-pdf, or html) and load it in addition to this resolution fragment.
5. If only metadata or an abstract is available, create a clearly labelled draft note from that material and record the unavailable sections. Do not fabricate body text.
